# Exp-009 — Progresión Inteligente (Fuerza + Cardio) (QAF)

Objetivo: que el usuario sienta:

- Progreso visible.
- Adaptación al día real (fatiga/estrés/sueño).
- Resolución de estancamiento sin riesgo.
- Seguridad (guardrails).

MVP (sin pantallas nuevas): el chat captura 1–2 ejercicios clave + RPE + % cumplimiento.

## Archivos

- `scripts/qaf_progression_intelligent.py`: motor QAF Exp-009.
- `scripts/run_local_eval.py`: runner JSONL.
- `data/dataset.sample.jsonl`: ejemplos.
- `tests/test_qaf_progression_intelligent.py`: tests unitarios.
