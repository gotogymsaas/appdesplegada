# Exp-013 — Posture & Proportion Intelligence™ (MVP)

Algoritmo unificado tipo coach:
- **Postura/alineación** (A-score)
- **Proporción (proxy por ratios)** (P-score)
- Índice unificado: **Alignment & Silhouette Index (ASI)**

Entrega:
- 2 correcciones inmediatas (30–60s)
- 1 ajuste semanal de entrenamiento

## Inputs (3 fotos)
- Frente relajado (obligatoria)
- Perfil derecho relajado (obligatoria)
- Espalda relajado (opcional recomendado)

Importante: la pose estimation ocurre en el **cliente** (MediaPipe Pose). El backend recibe solo **keypoints**.

## Guardrails
- No cm reales.
- No segmentación corporal.
- No diagnóstico médico.
- Proporción = eficiencia corporal (no estética).
