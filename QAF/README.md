# QAF (Quantum Analysis Framework)

Esta carpeta concentra la documentación y evidencia del experimento de desarrollo de algoritmos cuánticos bajo la metodología **Quantum Analysis Framework (QAF)**, sin mezclarla con `docs/` del producto.

## Estructura
- `00_inbox/`: documentos “raw” tal como llegan (PDF, DOCX, enlaces exportados, capturas). No editar aquí.
- `01_lecturas/`: notas en Markdown con lo que se extrajo de cada documento (resúmenes, glosario, decisiones).
- `02_experimentos/`: bitácoras y artefactos del experimento (pseudocódigo, resultados, scripts/notebooks si aplica).

## Recomendaciones para subir por GitHub
- Preferir **Markdown** (`.md`) para notas: fácil de revisar en PRs.
- Evitar subir **secretos** (tokens, llaves, strings de conexión) y **datos personales/sensibles**.
- Si un archivo pesa mucho (p. ej. >50–100MB), considerar **Git LFS**.
